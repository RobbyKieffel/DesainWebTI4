Review Game Mobile Legend










Mobile legends bang bang merupakan aplikasi permainan game moba 5v5 yang dapat dimainkan oleh pengguna di handphone ataupun pc.

Impresi awal kami terhadap game moba besutan Moonton ini cukup menarik, objektif game yang mudah dimengerti, pilihan hero yang bervariasi, dan permainan pertarungan tim yang cukup kompetitif. Sebagai game moba yang telah bertahan selama kurang lebih 5 tahun, tentunya game ini punya kelebihan dan kekurangan.



Dalam segi combat system, game ini sudah sangatlah umum seperti game moba pada umumnya. Pertarungan secara tim yang terdiri dari 5 anggota tim, yang memiliki misi untuk menghancurkan turet lawan.



Mekanik Game




Mekanik game ini sangatlah mudah, player di berikan kontrol analog untuk menggerakan hero. Ada juga ikon untuk mengeluarkan basic attack, skill 1, skill 2, ultimate, item shop, mini map, dan juga spell. Cara mainnya juga cukup mudah, player hanya perlu menggerakan analog untuk mengontrol hero yang kita pilih, kemudian gunakan basic attack, skill, dan juga spell di waktu yang tepat sehingga dapat mengalahkan lawan ataupun juga untuk menyelamatkan diri kita dari kejaran lawan.




Objective Game



Objective dalam game ini yang biasanya ada 3:

  • Menghancurkan turet lawan
  • menaikkan rank hingga mithical glory
  • menjadi pemain top untuk hero andalan



Platform OS & spesifikasi Minimum



Game ini telah saya mainkan sejak tahun 2017. Dan pada masa itu, bermain game ini menggunakan spesifikasi smarthphone dengan RAM 2GB masih lancar. Namun game ini bisa bertahan sampai saat ini pastinya karena pihak developer sering mengupdate game ini agar selalu diminati oleh pengguna. karena sering update, game ini pun tidak lagi layak untuk dimainkan di smartphone dengan spesifikasi RAM dibawah 4GB untuk OS Android.




Manfaat Game



Selain menikmati keseruan gamenya, game ini juga memiliki beberapa manfaat yakni:


  • Membuat pemainnya mengasah otak

    • Jangan heran mengapa Mobile Legend merupakan salah satu game yang memaksa player untuk memutar otak. Karena untuk memenangkan game ini, tidak mungkin hanya dengan menggerakan analog , membeli item, dan memencet skill atau spell secara asal-asalan. Untuk memenangkan game ini, kita harus menyusun strategi seperti, menentukan lokasi yang tepat untuk kita tuju, membeli item yang sesuai untuk hero kita, dan menggunakan skill pada waktu yang tepat.


  • Melatih Kemampuan Berkomunikasi

    • game multiplayer 5v5 online, sudah pasti game ini membutuhkan komunikasi antar sesama tim. Dalam game ini ada fitur untuk mempermudah kita dalam berkomunikasi dengan teman 1 tim kita. Ada fitur chat, dan juga voice chat.







  • Melatih Kerjasama/kekompakan dalam Tim

    • Walaupun dalam permainan kita sudah saling berkomunikasi tapi cara bermain tim kita belum kompak, tentu kita akan sulit untuk memenangkan permainan. Maka dari itu, dengan sering bermain bersama kita pasti akan semakin kompak dalam bermain game ini. Tidak menutup kemungkinan juga jika dalam kehidupan nyata kita juga bisa menjadi kompak dengan teman yang sering bermain dengan kita dalam game.


Walaupun game ini memiliki manfaat, namun jangan terlalu sering bermain game sampai lupa waktu. Kamu harus mampu mengatur waktu antara aktivitas anda sehari-hari dan juga waktu anda untuk bermain. jangan sampai hanya karena bermain game, anda lupa belajar, lupa kerja, bahkan lupa makan dan tidur.
